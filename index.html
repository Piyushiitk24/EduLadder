🎲 Rules for EduLadder Quest 🐍

▶ Starting the Game

  • ✶ A player must roll a 1 to start the game.
  • ✶ If the player rolls any other number, they must answer a question:
    • ✅ Correct answer → Game starts immediately.Player will get another chance to roll the dice.
    • ❌ Wrong answer → Turn passes to the next player.

🎲 Gameplay Mechanics

  • ✶ Players take turns rolling the dice.
  • ✶ The dice roll value determines how many steps a player moves.
  • ✶ If a player lands on a ladder, they move up.
  • ✶ If a player lands on a snake, they must answer a question:
    • Correct answer ✅ → Moves forward +2 (Bonus).
    • Wrong answer ❌ → Moves back to the snake’s tail (Penalty).

⭐Bonus points for Questions

  • ✶ If a player lands on a Question, a question will pop-up:
    • Correct answer ✅ → Moves forward +2. If landing on a snakebite +1.
    • Wrong answer ❌ → Moves Backward -2. If landing on a ladder -1.

🏆 Winning the Game

  • ✶ The first player to reach the last tile (100) wins.🏅
  • ✶ If a player exceeds the last tile, they missed the chance and shall wait for next turn.

📊 Scoring

  • ✶ The game tracks correct and incorrect answers per player.
  • ✶ The scoreboard is displayed at the end with following information:
    • Number of 🎲 moves to complete the game.
    • Total ✅correct answers & ❌ incorrect answers .
    • Number of 🪜 ladders used.
    • Number of 🐍 Snakebite encountered.
